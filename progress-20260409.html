遭遇問題與解決方式

問題：Claude API 回應中有時包含過多 Markdown 格式符號，導致前端顯示混亂。

解決方向：在 System Prompt 中明確要求以純文字或特定 JSON 格式回應，並在前端進行格式解析處理。

問題：長段程式碼貼入時，Token 用量超出預期，影響回應速度與成本。

解決方向：評估在前端加入程式碼長度限制提示，並研究 Claude API 的 Token 計算機制以進行最佳化。

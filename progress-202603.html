3月進度

我們針對專題所進行的一系列初步規劃與技術實踐過程 。這段時間的核心任務在於專題架構的深度討論，我們不僅多次交換意見以確認主題與名稱，更針對整體架構的可行性進行了嚴謹的評估與分析 。除了理論層面的探討，我們也同步展開了技術開發的基礎練習，重點介紹並導入了 Vercel 平台，並實際進行了簡易的功能測試 。為了強化實作經驗，我們進一步利用 Vercel 成功連結 GitHub 儲存庫，透過建立一個簡易的練習網站，親自操作並熟悉從程式碼託管到網站自動化部署的完整流程 。
